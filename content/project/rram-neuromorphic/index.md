### Project Overview
This project focuses on the integration of **Non-Volatile Memory (RRAM)** with neuromorphic computing architectures. 

**Industry Impact:**
- Reduced power consumption by 40% compared to traditional CMOS memory.
- Forming-free operation for flexible electronics.
- Direct applications in Edge-AI and Hardware Security (PUFs).
